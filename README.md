# Maharaj Portfolio

A modern, premium, highly interactive full-stack developer portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Customization

Replace placeholder files in `public/`:
- `profile.svg` → Your profile photo (update path in `Hero.tsx`)
- `certificate.svg` → Your certificate (update path in `Certifications.tsx`)
- Add `resume.pdf` for the download button

Update social links in `Hero.tsx`, `Contact.tsx`, and `Footer.tsx`.

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Or push to GitHub and import at [vercel.com](https://vercel.com).
